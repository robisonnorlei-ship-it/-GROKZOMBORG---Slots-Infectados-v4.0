ℹ️ SOBRE O VÍRUS

GrokZomborg Slots é um jogo de cassino zumbi cyberpunk altamente viciante, desenvolvido com pura criatividade e caos digital.

O jogo combina a emoção dos slots tradicionais com uma temática pós-apocalíptica de zumbis e vírus, criando uma experiência única e imersiva. Com símbolos virais (🧟 ☣ 🦠), multiplicadores insanos e a possibilidade de ganhar até 1000x sua aposta, GrokZomborg é o jogo perfeito para quem busca adrenalina digital.

⚙️ Tecnologia: HTML5 + CSS3 + JavaScript puro
🎮 Plataforma: Web (Desktop, Mobile, Tablet)
📦 Tamanho: Arquivo único HTML (~21KB)
🚀 Performance: Extremamente otimizado
❌ Dependências: NENHUMA
