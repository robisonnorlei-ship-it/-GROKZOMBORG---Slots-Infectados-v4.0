## 🆘 Troubleshooting

### Arquivo não abre
- [ ] Use navegador diferente
- [ ] Desbloquei arquivo (Windows)
- [ ] Baixei novamente
- [ ] Limpei cache (Ctrl+Shift+Del)

### Jogo está lento
- [ ] Feche abas desnecessárias
- [ ] Use Chrome/Firefox
- [ ] Desative extensões
- [ ] Reinicie navegador
- [ ] Reduza gráficos (Settings)

### Som não toca
- [ ] Aumente volume do navegador
- [ ] Confira settings do jogo
- [ ] Não está em mute?
- [ ] Reinicie navegador

### Dados desapareceram
- [ ] Verificar localStorage
- [ ] Backup foi deletado?
- [ ] Cache foi limpo?
- [ ] Navegador privado?

### Mobile travando
- [ ] Feche apps em background
- [ ] Limpe cache
- [ ] Reinicie dispositivo
- [ ] Atualize navegador
